## Details
- deep-river is a Python library for online deep learning. deep-river's ambition is to enable [online machine learning](https://www.wikiwand.com/en/Online_machine_learning) for neural networks. It combines the [river](https://www.riverml.xyz/) API with the capabilities of designing neural networks based on [PyTorch](https://pytorch.org/).
